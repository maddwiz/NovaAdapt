# Android Play Store Kit

This directory is the repo-owned launch kit for publishing `com.novaadapt.operator`.

Contents:

- `en-US/short-description.txt`
- `en-US/full-description.txt`
- `en-US/release-notes.txt`
- `privacy-policy/index.html`
- `data-safety.md`
- `release-checklist.md`

Use `scripts/package_play_store_kit.sh` to bundle these files with the latest built APK/AAB into a single artifact under `dist/`.
